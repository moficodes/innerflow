# Innerflow

Innerflow is a minimalist, guided breathing and meditation application designed to help high-performers achieve neural synchronization and deep flow states.

## Productivity Value
True productivity isn't just about output; it's about energy management. Innerflow provides structured breathing routines (like Box Breathing and 4-7-8) that quickly reset your autonomic nervous system. By taking 2-minute breathwork breaks, you can significantly reduce cortisol (stress), lower your heart rate, and return to deep work with enhanced focus and mental clarity. It includes session tracking to monitor your consistency.

## How to Run & Test
This project uses Vite and requires Node.js.

1. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`
2. Start the development server:
   \`\`\`bash
   npm run dev
   \`\`\`
3. Open the application, authenticate, and try the "Practice" tab to follow a guided breathing routine. Check your analytics to see your recorded sessions.
